Как играть онлайн?

Онлайн игра (P2P через WebRTC):

1. Создать комнату:
• Нажмите "Создать комнату"
• Скопируйте ваш ID
• Отправьте ID другу

2. Присоединиться:
• Нажмите "Присоединиться"
• Введите ID друга
• Подождите подключения

3. Играть:
• Соединение устанавливается напрямую
• Ходы синхронизируются автоматически
• Можно играть сколько угодно партий

Важно: Оба игрока должны быть онлайн одновременно!
